# Components — ts-ui

C4 Level 3 components for `ts-ui`.

| Component | Pattern                                      |
| --------- | -------------------------------------------- |
| Button    | CVA variants, 6 variants, 8 sizes, `asChild` |
| Alert     | CVA variants, `role="alert"`                 |
| Dialog    | Radix Dialog, portal, overlay, close button  |
| Input     | focus-visible, `aria-invalid`                |
| Card      | Subcomponents with `data-slot`               |
| Label     | Radix Label                                  |
