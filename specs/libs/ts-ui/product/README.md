# Product — ts-ui

C4 Level 1 product framing for `ts-ui`. See
[Specs Directory Structure Convention](../../../../repo-governance/conventions/structure/specs-directory-structure.md)
for the canonical layout.

## Overview

`ts-ui` is the shared React component library for the `ose-primer` monorepo, built on shadcn/ui
patterns with Radix UI primitives and Tailwind CSS. It provides the accessible, themeable building
blocks (Button, Alert, Dialog, Input, Card, Label) consumed by every TypeScript frontend in the
workspace.

See [overview.md](./overview.md) for the full product overview.
