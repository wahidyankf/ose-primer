# Repository Instructions for Amazon Q

This repository's canonical, vendor-neutral instructions for every contributor
and AI coding agent live in `AGENTS.md` at the repository root. Read it first —
it is the single source of truth.

Amazon Q does not yet read the root `AGENTS.md` natively, so this rules file
bridges the gap. It is generated by `rhino-cli agents emit-bindings`; do not edit
it by hand.
