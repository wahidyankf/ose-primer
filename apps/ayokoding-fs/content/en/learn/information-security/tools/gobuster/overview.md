Gobuster is a high-performance tool for directory/file brute-forcing, DNS subdomain enumeration, and virtual host discovery. Written in Go with concurrent processing for rapid enumeration.

## Available Content

- **[Quick Start](/en/learn/information-security/tools/gobuster/quick-start)** - Get started with directory brute-forcing and DNS enumeration (5-30% coverage)
- **[Beginner](/en/learn/information-security/tools/gobuster/beginner)** - Master all three modes and real-world enumeration scenarios (0-60% coverage)

## Key Features

- **Fast Performance** - Written in Go with concurrent processing
- **Multiple Modes** - Directory/file brute-forcing, DNS enumeration, virtual host discovery
- **Simple Interface** - Clean command-line with straightforward options
